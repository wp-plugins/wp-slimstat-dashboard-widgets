== Description ==
Add-on for WP SlimStat which enables admins to monitor their visitors directly from their Wordpress dashboard.

= Requirements =
* Wordpress 3.1 or higher (it may not work on large multisite environments; some users have reported problems in accessing the configuration page under Wordpress 3.3.x or earlier)
* [WP SlimStat 2.8+](http://wordpress.org/extend/plugins/wp-slimstat/)
* PHP 5.1 or higher
* MySQL 5.0.3 or higher

= Browser Compatibility =
WP SlimStat Dashboard Widgets uses the HTML5 Canvas element to display its charts. Unfortunately Internet Explorer 8 and older versions don't support this functionality, so you're encouraged to upgrade your browser.

== Installation ==

1. Go to Plugins > Add New
2. Search for WP SlimStat Dashboard Widgets
3. Click on Install Now under WP SlimStat Dashboard Widgets
